NoxBit - Meet the New Generation of Live Video Streaming!

Quick start:

    Available kits:

    /AArch64/noxbit-pa-ARM-64.tar.gz
    /MIPSel/noxbit-pa-MIPSel-32.tar.gz
    /ARMel/noxbit-pa-ARMel-32.tar.gz
    /ARMhf/noxbit-pa-ARMhf-32.tar.gz
    /i386/noxbit-pa-x86-i386.tar.gz


 0. Download selected kits from  http://download.noxbit.com, for example:
    wget http://download.noxbit.com/i386/noxbit-pa-x86-i386.tar.gz

 1. Unpack noxbit-pa-....tar.gz into the /opt/noxbit
 2. Edit /opt/noxbit/noxbit.cfg file with your UID and password
 3. Start NoxBit peer agent by :
	/opt/noxbit/STM-Hypervisor -config=/opt/noxbit/noxbit.cfg

Enjoy!
